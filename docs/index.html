Why Lux

Everything you need. Nothing you don't.

Purpose-built for SaaS dashboards, admin panels, and AI products. Ships production-quality components without the framework tax.

Semantic first

Style native HTML elements — buttons, inputs, dialogs. No div soup or class overload.

Zero dependencies

Pure CSS and vanilla Web Components. No React, no build pipeline required.

Dark mode native

Uses CSS light-dark() — auto-follows system preference with zero JS.

CSS custom properties

Every value is a token. Override --brand and the whole system recolors.

Accessible by default

ARIA roles, keyboard navigation, and focus management built into every interactive component.

AI SaaS aesthetic

Zinc palette, glass cards, subtle gradients — the design language of Linear, Vercel, and Raycast.
